HireHunt is a web-based platform designed to help students track and manage job and internship applications in one place.

## Features
- User authentication (login & signup)
- Explore and save job opportunities
- Search and filter jobs
- Track application status (Saved, Applied, Interview, Offer, Rejected)
- Add personal notes for each application
- Delete saved applications
- Dashboard with real-time application statistics

## Tech Stack
- React (Vite)
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore

## How to run locally
cd client
npm install
npm run dev

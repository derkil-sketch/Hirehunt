HireHunt is a modern, student-focused web application designed to help users organize, track, and manage their job and internship applications in one centralized platform. It simplifies the placement and internship process by providing a clean dashboard and an easy-to-use tracking system.

---

## ‚ú® Features

- Secure user authentication (Login & Signup)
- Explore available job and internship opportunities
- Save jobs for later tracking
- Search and filter jobs by role, company, and location
- Track application status  
  *(Saved, Applied, Interview, Offer, Rejected)*
- Add personal notes for each application
- Delete saved applications
- Dashboard with real-time application statistics
